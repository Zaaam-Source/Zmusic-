Fitur

Production-ready,
bukan template biasa

MVVM + Hilt, ~12.000 baris Kotlin, banyak edge case sudah dihandle & terdokumentasi.

🎵

YouTube Streaming

Via NewPipe Extractor tanpa API key berbayar. Gratis selamanya.

💜

Mood Detection AI

6 mood dengan weighted keyword scoring + history-based detection + smart shuffle.

🫧

Floating Bubble

Overlay player bisa di-drag, tetap aktif di atas aplikasi lain.

📊

Spotify Wrapped

Rekap per periode: top songs, artis, waktu dengar, distribusi mood.

📥

Download & Offline

Download ke storage lokal, playback otomatis dari file lokal.

⏱️

Sleep Timer & Speed

Sleep timer, speed control, share Now Playing Card ke WA/IG.

🎛️

Audio Visualizer

Real-time dari AudioSession ID dengan fallback animasi otomatis.

📅

Daily Mix Generator

Playlist harian otomatis dari histori user + trending YouTube.

📚

Playlist CRUD

Buat, edit, hapus playlist. Semua tersimpan di Room DB lokal.

Tech Stack

Teknologi modern,
kode bersih

Semua library stable release — tidak ada dependency eksperimental.

🟣 Kotlin
🎨 Jetpack Compose
🎵 ExoPlayer / Media3
💉 Hilt (DI)
🗄️ Room Database
📡 NewPipe Extractor
🔄 Coroutines + Flow
🏗️ MVVM Architecture
🖼️ Coil
🧭 Navigation Compose
📳 MediaSession
🔒 FileProvider
